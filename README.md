# PACIFIC SHIELD

現代の実在艦名を使用した、ブラウザ向け艦隊戦術ゲームの試作版です。多国籍合同演習を舞台に、BLUE CELLの艦船を指揮してORANGE CELLを無力化します。

## 起動

```bash
npm install
npm run dev
```

表示された `http://127.0.0.1:5173/` をブラウザで開きます。

## 作戦と難易度

起動時の作戦ブリーフィングで、次の3シナリオと難易度を選択できます。

- 北太平洋制海演習：ORANGE CELL全艦を無力化
- 空母打撃群防衛：HMS Queen Elizabethを撃破し、USS Gerald R. Fordを生還
- 深海ハンター：ROKS Dosan Ahn Changhoを無力化し、JS Oryuを生還

難易度は「士官候補生」「指揮官」「提督」の3段階です。高難易度ほど敵攻撃の命中率と威力が上がり、警報から着弾までの猶予が短くなる一方、最終得点倍率が上昇します。シナリオごとに敵航空隊・ミサイル・魚雷の攻撃頻度と勝利条件が変化します。

## 戦績・勲章・艦隊強化

任務終了時に得点、勝敗、経験値、補給ポイント、勲章がブラウザへ自動保存されます。経験値に応じて階級が上がり、初勝利、精密射撃、艦隊防空、対潜戦、航空優勢、完全防衛の条件を達成すると勲章を獲得します。

初回配備予算として2補給ポイントが付与され、作戦ブリーフィングの「FLEET UPGRADES」で使用できます。射撃管制は全兵装の命中率、艦隊防空は迎撃弾・電子戦・音響囮、損傷統制は消火・排水・応急修理能力を恒久的に強化します。各系統は最大Lv 3です。保存データにはバージョン番号を付け、破損または非対応データは安全な初期値へ戻します。

## 北太平洋キャンペーン

「北太平洋戦役」は5作戦で構成され、前の作戦に勝利すると次の作戦が解放されます。各作戦は使用シナリオ、難易度、得点ボーナスが固定され、最終評価に応じて1～3個の星を獲得します。再挑戦で上位評価を得ると最高の星数へ更新されます。

1. CP-01 氷海の槍
2. CP-02 空母の盾
3. CP-03 静かなる潮流
4. CP-04 鋼鉄の嵐
5. CP-05 太平洋の冠

キャンペーン作戦には追加の補給ポイント報酬があります。全作戦の完遂で「太平洋戦役章」を獲得し、キャンペーン進捗と星評価は通常の戦績と一緒に自動保存されます。

## 戦術訓練・操作ガイド

初回の作戦開始時には6段階の戦術訓練が自動的に開きます。指揮艦と目標の選択、艦隊機動、兵装、F-35C航空隊、防空・対魚雷、対潜戦・損傷統制を順番に確認できます。訓練中は戦術時間が停止し、閉じると作戦へ戻ります。

訓練の完了状態は戦績データへ保存されるため、2回目以降は自動表示されません。上部の「操作ガイド」または作戦ブリーフィングのガイドボタンから、いつでも再表示できます。

## 操作

- 艦船一覧または海上の艦船をクリック：指揮艦を選択
- 海上のORANGE CELL艦またはレーダー上の赤い目標をクリック：攻撃目標をロック
- 空いている海面をクリック：艦隊の移動地点を指定
- 「単縦陣」「輪形陣」：艦隊の隊形を変更
- 速力ボタン：速力を変更
- `−` / `＋`、または `A` / `D`、左右矢印：針路を10度変更
- 兵装を選択して「発射」、または `Space`：最寄りの対抗艦を攻撃
- `P` または「一時停止」：戦術時間を停止・再開
- ミサイル警報時に「迎撃」または `I`：迎撃ミサイルを発射
- ミサイル警報時に「電子戦」または `E`：敵ミサイルの命中率を低下（EWゲージを25%消費）
- ミサイル警報時に「デコイ」または `C`：チャフ／フレアを展開して命中率を低下
- 「CAP発艦」または `L`：CVN-78からF-35C 2機を発艦させ、防空哨戒を実施
- 「対艦発艦」または `K`：F-35C 4機を識別済み目標への対艦攻撃に投入
- 敵航空隊接近時の「艦対空」または `I`：迎撃ミサイルでF-35B編隊を削減
- 敵航空隊接近時の「電子戦」「回避支援」：攻撃精度を低下
- 「潜望鏡」「深度潜航」または `Z`：SS-511の深度を切り替え
- 「静粛航行」または `X`：速力を抑えて敵からの被探知率を低下
- 「アクティブソナー」または `Q`：周辺海域へPINGを送信し、敵潜水艦の音紋を識別
- 魚雷警報時の「音響囮」：有限のデコイを使用して魚雷の命中率を低下
- 「損傷訓練」：選択艦に模擬火災・浸水を発生させ、損傷統制を訓練
- 「消火」「排水」「応急修理」：資材を消費して火災・浸水・船体損傷を復旧
- 選択したシナリオの任務目標を達成すると演習完了

兵装ごとに射程と命中率が異なります。目標パネルで距離と命中予測を確認し、対艦ミサイル・航空攻撃・魚雷などを使い分けてください。命中、目標無力化、射撃精度が演習得点と最終評価に反映されます。

敵艦は味方艦のレーダー・ソナー探知圏に入るまで表示されません。探知圏の外縁では艦名を伏せた「未確認目標」として表示され、艦隊を接近させると識別・攻撃できるようになります。損傷が重大な艦は探知距離も低下します。

ORANGE CELLは一定間隔で対艦ミサイルを発射します。警報から着弾までの短い時間に、確実性の高い迎撃、命中率を下げる電子戦、有限のデコイを組み合わせて防御してください。各防御資源と戦果は「FLEET DEFENSE」に表示されます。

ORANGE CELLの各艦は、BLUE CELLの艦種・損傷・距離を評価して高価値目標を選択します。敵艦は状況に応じて接近、側面機動、攻撃位置の維持、被攻撃時の回避、重大損傷時の離脱を行います。識別済み目標の推定行動と攻撃対象は「TARGET LOCK」で確認できます。

CVN-78のVFA-147所属F-35Cは、CAPと対艦攻撃の2任務に投入できます。航空隊は発艦、哨戒または攻撃進出、攻撃、帰投、再武装を順に行います。CAP展開中は敵ミサイルを早期に妨害して命中率を低下させます。航空隊の機数、燃料、状態、戦果は「AIR WING」で確認できます。艦載機はF-35C兵装として選択した場合も、即時攻撃ではなく同じ発艦サイクルを開始します。

HMS Queen ElizabethはF-35B 6機編隊を高価値艦へ投入します。事前にCAPを展開していればVFA-147が自動交戦し、敵編隊数と攻撃精度を低下させます。CAPがない場合も「FLEET DEFENSE」から艦対空迎撃、電子戦、回避支援を組み合わせられます。残存機を許した場合は航空攻撃による船体損傷・火災・浸水が発生します。

JS Oryuは潜望鏡深度と深度潜航を切り替え、静粛航行で音響シグネチャを抑えられます。深度潜航と静粛航行は敵AIから狙われにくくなる一方、速力が制限されます。通常は隠れているROKS Dosan Ahn Changhoをアクティブソナーで識別すると、SS-511と89式魚雷が自動選択され対潜攻撃へ移れます。敵潜水艦の魚雷には迎撃ミサイルや電子戦を使用できないため、音響囮で回避してください。

被弾すると、兵器種に応じて火災と浸水が発生します。放置した火災・浸水は継続的に船体を損傷し、最高速力、センサー探知距離、兵装命中率を低下させます。「DAMAGE CONTROL」で選択艦の戦闘効率を確認し、有限の消火剤・排水能力・予備部品を配分してください。空母の火災・浸水が重大になると、飛行甲板も使用不能になります。

## 収録艦

USS Gerald R. Ford、USS Arleigh Burke、JS Maya、FS Aquitaine、JS Oryu、HMS Queen Elizabeth、HMS Daring、HMAS Brisbane、FS Alsace、CNS Nanchang、ROKS Dosan Ahn Changho。

艦船・海面画像は本プロジェクト向けに生成したゲーム素材です。実艦の精密な3Dモデルではなく、各艦種の視認性を優先した試作表現です。
